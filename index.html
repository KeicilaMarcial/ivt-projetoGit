Criando repositório

Iniciando git local

git init

Adicionando repositório remoto

git remote add origin [uri-repositorio.git]

Clonando repositório

git clone [uri.git]

Adicionando arquivos

git add . ou git add [nome-arquivo]

Commitando arquivos

git commit -m "[mensagem-descritiva]"

Subindo commit para o servidor

git push origin [branch]

Atualizando repositorio local

git pull origin [branch]

Resolvendo conflitos

Conflitos ocorrem quando duas ou mais alterações são feitas na mesma linha do mesmo arquivo, para resolver é necessário analisar quais alterações devem permanecer e quais devem ser descartadas. Isso pode ser feito via GUI ou editor de texto.
